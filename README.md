# Sum Api Terraform

Terraform infrastructure for Sum Api project.

## Getting Started

### Prerequisites

```
Terraform
Hashicorp AWS 3.0
Create AWS profile "tenpo"
```

### Installing

terraform init


## Deployment

```
terraform plan
terraform apply
```

## Built With

* [Terraform](https://www.terraform.io/) - IaC tool
